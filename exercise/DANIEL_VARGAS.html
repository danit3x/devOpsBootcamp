Examen Git y Github

Alumn@: [Daniel Vargas]

  1. Que es un sistema de control de versiones?

    [Es un sistema que nos permite llevar un seguimiento de los cambios que hacemos en nuestros archivos
    y en caso de requerirlo, poder volver a un estado anterior y recuperar cambios o archivos]

  2. Cual es la diferencia entre git y github?

    [Git es la aplicacion del control de versiones,
    Github es la plataforma social donde se publican los repositorios que pueden
    ser utilizados para trabajo remoto en equipos]

  3. Cómo creo una rama en git?

    [Situandonos en la rama a replicar, utilizamos el comando
    git checkout -b nombreDeLaRama]

  4. Cómo clono un repositorio?

    [Con el comando git clone urlRepositorioGithub]
